Welcome to ORANSim's Documentation!
====================================

ORANSim is an open-source simulation library for 5G Open Radio Access Networks (ORAN). It is designed to be a powerful and extensible tool for researchers and engineers working on various aspects of 5G ORAN, with a particular focus on AI/ML-driven optimization and automation.

This documentation provides a comprehensive guide to using and extending ORANSim.
